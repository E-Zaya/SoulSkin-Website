## デザインのこだわり

Soul Skinのブランドサイトでは、ただシンプルに整えるのではなく、静けさの中に少しの緊張感や違和感が残るデザインを意識しました。黒を基調にしたミニマルな構成をベースに、余白の取り方、タイポグラフィの強弱、写真の配置バランスを細かく調整し、若者向けストリートブランドらしい空気感が伝わるように設計しています。


また、noise表現や控えめなアニメーションを加えることで、単調にならず、スクロールするたびに印象が残る体験を目指しました。情報量を増やしすぎず、ロゴ・写真・文字の見せ方そのものでブランドの世界観を表現した点が、このサイトの大きなこだわりです。
